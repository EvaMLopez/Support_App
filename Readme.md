# CRUD APP

SpringBoot project with dependencies:
-  H2
-  Spring Data JPA
-  Spring Web
-  Spring Boot Dev tools


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compile and Minify for Production
```
npm run build
```
